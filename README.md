# T9 Dictionary

## Use

### Install

```
$ git clone <repo> && cd <repo>
$ npm install
```

### Run

```
$ npm start
```

Should open the browser automatically, if not navigate to `localhost:8080`

### Test

```
$ npm test
```

Runs jest in watch mode
